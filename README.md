# Active Directory Enterprise Lab

## Overview

This project documents my enterprise Active Directory home lab, designed to simulate a production Windows environment for practicing identity management, enterprise administration, and security operations.

The lab focuses on the technologies and workflows commonly used by enterprise IT and Security Operations teams.

---

## Objectives

- Deploy Active Directory Domain Services (AD DS)
- Configure Organizational Units (OUs)
- Create users and security groups
- Configure DNS and DHCP
- Implement Group Policy Objects (GPOs)
- Practice PowerShell administration
- Apply least privilege principles
- Simulate enterprise identity management

---

## Technologies

- Windows Server
- Active Directory
- DNS
- DHCP
- Group Policy
- PowerShell
- Hyper-V / VMware Workstation
